Projects

TicTacToe Game

Designed and implemented a Tic Tac Toe game with a graphical user interface (GUI) using Java and object-oriented programming principles. The game features a sign-in page where users enter their name and ID, with the information stored in an MS Access database via the UCanAccess driver. After successful sign-in, users are greeted with a welcome page displaying a "Welcome to Tic Tac Toe Game" message and a start button to begin the game. This project highlights skills in Java programming, database integration, and user interface design, demonstrating the ability to create interactive and engaging applications.

Learning Management System

Developed an LMS (Learning Management System) website utilizing HTML, CSS, and JavaScript. The platform offers a user-friendly interface for both students and educators, featuring functionalities such as user authentication, course management, assignment submission, and progress tracking. This project exemplifies expertise in front-end web development, enhancing user experience, and creating dynamic, interactive educational environments.

Web Portal

Created a comprehensive management system using C# to manage students, teachers, and administrators. The system features a user-friendly interface that facilitates user authentication, data management, role-based access control, and interactive dashboards. This project highlights proficiency in C# and web development, demonstrating the ability to create dynamic, responsive, and efficient management systems tailored to educational institutions.
